* [4mm Binding posts ](https://www.amazon.com.au/dp/B0G87LDQ7T)

Pi Specific  
* [GeeekPi Armor Lite V5](https://www.amazon.com.au/dp/B0CNVFCWQR)

* [Hifiberry Amp4 Pro](https://www.hifiberry.com/shop/boards/amp4pro/)
